# C Programming Practice

This repository contains C programs I wrote while learning C in PU and 1st year B.Tech CSE.  
It is an archive of my early practice covering core language concepts.

## 📂 Topics covered

- Basics: input/output, operators, simple calculations
- Control flow: `if` / `else`, `switch`, loops
- Arrays and strings: 1D/2D arrays, basic string handling
- Functions: parameter passing, return values
- Other small exercises and lab-style programs

## 🔧 Tech

- Language: C
- Tools: GCC, VS Code

```bash
# clone the repo
git clone https://github.com/dhanu0304/C-Programming_codes.git
